# Experiment Code Notes

These files are archived execution artifacts from the original experiments.

- They are included as evidence of the actual implementation work.
- Some scripts still use `/workspace/...` paths because they were run on remote training pods.
- They are not intended to be a polished standalone package in this repository.

The primary files for evaluating the research claims are in:
- `../results/`
- `../docs/`

The code is included so a reader can inspect how the experiments were wired and what implementation changes were required.
